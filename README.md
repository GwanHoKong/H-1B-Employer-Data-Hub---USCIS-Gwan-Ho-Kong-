## Project Overview

This project analyzes patterns in **H-1B visa sponsorship and approval outcomes** using employer-level data from the USCIS H-1B Employer Data Hub.

The analysis focuses on identifying:

- Companies that sponsor the most H-1B workers  
- Industries with the highest number of H-1B approvals  
- U.S. states with the largest number of H-1B cases  
- Denial rates across different petition types  

The analysis uses the **FY2025 Employer Information dataset**, which contains about **62,000 employer-level records**.

---

## Dataset

Source: USCIS H-1B Employer Data Hub

Key variables include:

- Employer (Petitioner) Name  
- Industry (NAICS Code)  
- Petitioner City / State / ZIP  
- Petition type approvals and denials  
- Fiscal Year
